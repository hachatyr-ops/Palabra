
# Palabra - Spanish Learning App

A minimalist and efficient tool for learning Spanish vocabulary with AI-powered mnemonics and pronunciation.

## Deployment on Vercel

1. Push this code to a GitHub repository.
2. Import the project in [Vercel](https://vercel.com).
3. In the **Environment Variables** section, add:
   - `API_KEY`: Your Google Gemini API Key.
4. Click **Deploy**.

## Features

- **AI Translation**: Instant Spanish to Russian translation using Gemini 3 Flash.
- **AI Mnemonics**: Get clever memory hints to remember difficult words.
- **Voice Support**: High-quality Spanish pronunciation (TTS).
- **Offline First**: All your words are saved in your browser's local storage.
- **Import/Export**: Easy backup and migration of your dictionary.
